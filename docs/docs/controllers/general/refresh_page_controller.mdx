## Purpose

Refresh the user's page, either when the controller comes into the DOM, or in response to another action.


## [Actions](https://stimulus.hotwire.dev/reference/actions)

| Action    | Purpose                                             |
|-----------|-----------------------------------------------------|
| `refresh` | Refreshes the user's page using `location.reload()` |

## [Values](https://stimulus.hotwire.dev/reference/values)

| Value    | Type    | Description                                                           | Default |
|----------|---------|-----------------------------------------------------------------------|---------|
| `onLoad` | Boolean | Set to `true` to immediately refresh the page on controller `connect` | `false` |
